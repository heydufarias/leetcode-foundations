# leetcode-foundations

Progressive Dart exercises to build a solid foundation in LeetCode logic patterns.

## Why this exists

LeetCode difficulty is often not related to syntax, but to the lack of training in underlying problem-solving patterns. 
This repository isolates those patterns into small, focused exercises, each serving as a building block toward solving interview-style problems.

## What LeetCode actually tests

Problems typically do not require the explicit use of constructs like loops, but instead the recognition and application of specific patterns. 
The most common patterns at Easy and Medium levels include:

- **Two pointers** - two indices working together on the same or different lists
- **Sliding window** - a window that expands and contracts over a list
- **Fast & slow** - two pointers moving at different speeds
- **Merge pattern** - two pointers, one per list, building a result together
- **Binary search** - eliminating half the search space each step
- **Prefix sum** - accumulating a previous result to answer range queries

## How to use

Each exercise should be solved without consulting the answer key. Answer keys are provided at the end of each module file. 
Progression to the next block is recommended only after the current one can be solved independently.
